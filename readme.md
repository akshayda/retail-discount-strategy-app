## Getting Started

### <p>Prerequisite</p>
<pre>
<code>$ node -v
v10.x.x</code>
</pre>

### <p>Download the package and Run command to install packages:</p>
<pre>
<code>$ npm install</code>
</pre>

### <p>To run test:</p>
<pre><code>$ npm run test</code>
</pre>

### <p>To generate Code Coverage:</p>
<pre><code>$ npm run test-coverage</code>
</pre>
